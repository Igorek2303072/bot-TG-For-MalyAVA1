# 🤖 CS2 AI News Bot

Telegram-бот для автоматической генерации и публикации новостей по Counter-Strike 2 (CS2) в канал. Бот собирает свежие статьи из популярных киберспортивных RSS-лент, переписывает их в короткие и живые посты с помощью нейросети **DeepSeek** и отправляет администраторам на модерацию (кнопки одобрения/перегенерации).

## ✨ Возможности

- 📡 **Сбор новостей:** Автоматический парсинг RSS-лент (HLTV, Escorenews, Cybersport, Esports).
- 🧠 **AI-генерация:** Использование DeepSeek (через OpenAI API) для создания уникального и читабельного текста без кликбейта.
- 🖼️ **Парсинг картинок:** Автоматический поиск и извлечение обложек статей со страниц источников (OpenGraph, Twitter Cards).
- ⚙️ **Система модерации:** Inline-кнопки для проверки поста перед публикацией (✅ Отправить в канал, ❌ Отменить/Сгенерировать новый).
- 🏷 **Авто-брендирование:** Бот гарантированно добавляет заданный тег (например, `@malyAVA1_twitch_bot`) в конец каждого поста.
- ✍️ **Ручной режим:** Возможность отправить свой собственный текст и картинку напрямую через бота.
- 🔐 **Безопасность:** Доступ к командам есть только у авторизованных администраторов. Чувствительные данные (токены, ключи) спрятаны в `.env`.

## 🛠 Стек технологий

- **Python 3.9+**
- **python-telegram-bot** (v20+) — взаимодействие с Telegram API.
- **OpenAI API** — интеграция с LLM DeepSeek.
- **BeautifulSoup4 & Requests** — парсинг веб-страниц и мета-тегов для картинок.
- **Feedparser** — чтение RSS-лент.
- **python-dotenv** — безопасное управление ключами и настройками.
